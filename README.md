# TaskMaster IT - Gestor de Tareas por Prioridad

TaskMaster IT es una aplicación web minimalista diseñada para la gestión eficiente de tareas académicas y personales.  

A diferencia de un tablero Kanban tradicional, organiza el flujo de trabajo basándose en niveles de **prioridad (Alta, Media, Baja)** y el control de **fechas límite**, permitiendo enfocarse en lo realmente urgente.

---

## Tecnologías Utilizadas

- **HTML5** → Estructura semántica
- **CSS3** → Diseño responsivo (Grid + Flexbox + Variables CSS)
- **JavaScript (Vanilla)** → Manipulación del DOM en tiempo real
- **IA Generativa (Gemini)** → Apoyo en diseño, lógica y debugging

---

## Uso de Inteligencia Artificial

Este proyecto fue desarrollado mediante **Pair Programming con IA**, donde la IA actuó como asistente en distintas etapas:

### Aplicaciones de la IA

1. **Arquitectura del diseño**
   - Uso de variables CSS (`:root`)
   - Sistema de colores consistente

2. **Refactorización**
   - Cambio de estados → sistema de prioridades

3. **Mejora UX/UI**
   - Márgenes laterales (10%)
   - Navbar fija
   - Alineación de formulario con tablero

4. **Debugging**
   - Corrección de errores entre HTML y JavaScript
   - Ajuste de selectores e IDs
