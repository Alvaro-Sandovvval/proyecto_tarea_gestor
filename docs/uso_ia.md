* Gestor de Tareas IT - TaskMaster
Este proyecto es un Gestor de Tareas estilo Kanban desarrollado para la asignatura de Herramientas Ofimáticas para la Gestión. La aplicación permite organizar tareas por niveles de prioridad (Alta, Media y Baja) e incluye un registro de fechas límite para una mejor gestión del tiempo.



*Uso de Inteligencia Artificial (Gemini)
Para el desarrollo de este proyecto, se utilizó IA (Gemini) como un colaborador técnico (Pair Programming). La IA no solo generó código, sino que ayudó en la arquitectura del diseño y la resolución de problemas lógicos.

*¿En qué me ayudó la IA?
Estructuración de Código: Transformar una lista de requisitos académicos en una estructura de archivos real (HTML, CSS, JS).

Diseño UI/UX: Creación de una interfaz moderna basada en variables de CSS, con un diseño adaptable (responsive) y una barra de navegación fija.

Lógica de Programación: Desarrollo de un script en JavaScript que captura datos de un formulario y los organiza dinámicamente en columnas específicas sin recargar la página.

Refactorización: Ajustar el diseño inicial para pasar de un sistema de "Estados" (Pendiente/Progreso) a un sistema de "Prioridades" (Alta/Media/Baja) a 
petición del usuario.



* Prompts Utilizados (Ejemplos)
A continuación, se detallan algunos de los comandos o "prompts" clave que se utilizaron para guiar a la IA durante el proceso:

Diseño Inicial: "Mejorame el diseño de mi página es un gestor de tareas... (requisitos de variables CSS y colores)".

Mejora de Interfaz: "Ponle navbar, y que el formulario use el mismo ancho que el de la tabla de las tareas, y pon mas margenes en la pagina en los lados".

Funcionalidad Específica: "En vez de estados, debería decir prioridad: alta, media y baja, y también falta la fecha límite de las tareas".

Corrección Técnica: "Haceme el script de nuevo ya que este no me funcionó con lo nuevo que edité... (para sincronizar IDs de HTML con JS)".



* Tecnologías Utilizadas
HTML5: Estructura semántica de la aplicación.

CSS3: Diseño basado en Flexbox y Grid, con uso intensivo de variables nativas.

JavaScript (Vanilla): Manipulación del DOM para la creación dinámica de tareas.

IA: Gemini (Google) para asistencia en codificación y documentación.
